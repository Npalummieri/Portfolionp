Proyecto -Pet Hero-

Pagina web dedicada a la contratacion de cuidado de mascotas,con registro de usuarios por roles donde cada uno desempeña un papel definido,teniendo CUIDADORES y DUEÑOS como protagonistas del mismo. Ahora mismo está observando la pagina principal que se encuentra al ingresar a la web. Se ven pequeños perfiles de cada cuidador con una serie de caracteristicas a partir de allí el usuario tiene distintas formas de interactuar con el sitio.

Vista del perfil de Cuidador estando conectado como dueño. La web posibilita visualizar a detalle la disponibilidad y caracteristicas de cuidado de la mascota. El dueño puede redactar una review con un puntaje para medir la reputación del cuidador y que sirve para futuros 'clientes'.

Mascotas del dueño conectado a la web.Tiene la posibilidad de editar sus datos y borrarlo según el caso.Tambien se manejan las subidas de archivo para las imagenes y videos de la mascota (plan de vacunacion y foto de perfil,por ejemplo).

Pagina de ingreso para cualquier tipo de usuario,con reestablecimiento de contraseña por mail en caso de olvido/perdida.

Vista de reserva a realizar con fechas especificas y el muestreo de un listado de las mascotas propias que coincidan con el perfil del cuidador.

Chat entre partes que confirmen una reserva para mejor comunicación.

Herramientas

PHP

PHP es un lenguaje de programación interpretado del lado del servidor y de uso general que se adapta especialmente al desarrollo web. Formó parte del temario para la finalización de la carrera y empleado en el proyecto final de la misma.

JAVA

Lenguaje de programación orientado a objetos. Aprendido durante la fase media de la carrera con enfoque en el paradigma de la programación orientada a objetos

C

Lenguaje de programación estructurado,de objetivo general.Aplicado durante la primera parte de la carrera sirvió como base para entender los fundamentos de los distintos tipos de datos y su manejo de forma general.

JavaScript

Lenguaje de programación compilado orientado a objetos.Utilizado como herramienta para el manejo dinámico de la web, tambien se usó en menor medida unos de sus más famosas librerias Jquery,una versión simplificada pero con el mismo objetivo de darle dinamismo a la web (aplicado junto con AJAX).

HTML

Lenguaje de marcado para la creación de paginas web.Sirve como estandar para la presentacion contenido en nuestra web.Lo utilicé en todo proyecto web mencionado en este portfolio.

CSS

Lenguaje de diseño grafico para darle una forma y estilo personalizado al documento de marcado (HTML). Utilizado en todos los proyectos web que he hecho junto con frameworks que utilizan de base a este (Bootstrap y aprendiendo Tailwindcss)

MySQL

Sistema gestor de base de datos relacional.Es la base del proyecto llamado "PetHero". Este sistema permite almacenar,organizar y recuperar datos tambien su tratado en terminos de rendimiento,seguridad y flexibilidad por ende posibilita el manejo de grande volumenes de los mismos de forma prolija.

GIT

Software de versionado utilizado en practicamente todos mis proyectos para llevar un registro de versiones del codigo. Muy eficiente para seguimiento de los archivos ,modificaciones ,agregacion y borrado involucrados en el trabajo.
